# RepoScope submission kit

This folder is the source of truth for the five-minute Qwen Cloud Global AI
Hackathon presentation.

## Assets

- [`five-minute-demo.md`](five-minute-demo.md) — timed narration, live-demo
  actions, fallbacks, and rehearsal checkpoints.
- [`submission-copy.md`](submission-copy.md) — copy-ready title, summary,
  description, track fit, technical highlights, and limitations.
- [`slides.html`](slides.html) — seven-slide, keyboard-controlled 16:9 deck.
- [`asset-checklist.md`](asset-checklist.md) — capture list and final submission
  QA checklist.

## Core message

**RepoScope turns one AI opinion into an inspectable engineering debate.** Four
Qwen agents inspect a repository through bounded tools, challenge opposing
positions, revise, and produce a schema-validated consensus with exact event
lineage.

## Voice

- Confident, not inflated.
- Technical, but understandable in one hearing.
- Evidence-led: show the event, file, tool call, or deployment proof.
- Honest about evaluation: the benchmark harness is reproducible, but its
  current two-trial run does not establish a quality advantage for debate.
